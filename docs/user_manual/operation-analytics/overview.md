## 总览页面详细说明

!!! Abstract " "
    -   云主机：统计所有云账号下云主机的数量；
    -   云磁盘：统计所有云账号下云磁盘的数量；
    -   宿主机：统计所有私有云账号下宿主机的数量；
    -   存储器：统计所有私有云账号下存储器的数量；
    -   云主机分布：统计所有云账号下云主机的分布数量；支持按组织、工作空间展示；
    -   云主机趋势：汇总云主机总数量趋势，支持按近7天、近30天、近半年、近一年展示；
    -   基础资源分配情况：统计私有云账号下宿主机的CPU分配率、内存分配率、存储器分配率；
    -   资源优化建议：根据设置的各纬度优化策略统计需要优化的资源数量。
